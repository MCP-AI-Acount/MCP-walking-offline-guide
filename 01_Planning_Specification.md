# 01 Planning Specification — walking-offline-guide

> fork: `italy-offline-guide` · 범용 도보 여행 오프라인

## 1. 주제

**도보 여행 어플 (오프라인)** — WiFi에서 여행 지역·POI·타일을 다운로드하고, 현장에서 GPS(또는 시뮬레이션)로 오프라인 지도·도보 안내.

## 2. UI 플로우

1. 로딩 + WiFi 경고 + 당일 도시 설명(로딩 화면만)
2. (비WiFi) 확인/나가기 → 세계지도 허브
3. 허브: 다운로드 완료 지역만 표시 · 방문=밝음 / 미방문=어두움
4. 여행 스케줄 짜기 → 1·2단계 설정 → 다운로드(%·취소·부분 유지)
5. 도시 선택 → 맵 화면(Italy 엔진 포팅 · 세로)

## 3. v1 범위

| 포함 | v1 |
|------|-----|
| 런타임 타일·POI 다운로드 | ✅ |

## 4. 완료 기준

1. `bash EXE/run_walking_guide_apk.sh` → debug APK
2. WiFi에서 도시 입력 → 다운로드 → 허브에 표시 → 맵 진입
